# Favourite Places
Kata APP to lookup addressess using the address.io api, and store them via localstorage. Electron, with angular 1.6 for routes / views. Webpack2, inuit css and karma for testing.

##Install dependencies
```js
npm install 
```
##Run Electron
```js
npm start
```
##Run Webpack
```js
npm run dev
```

##Build
```js
npm run build
```

##Test
```js
npm run test
```
